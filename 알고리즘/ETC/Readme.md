# 코딩테스트 라이브러리 정리

* itertools: 파이썬에서 반복되는 형태의 데이터를 처리하기 위한 유용한 기능들을 제공

     => 순열, 조합 알고리즘에 유용함

* heapq: Heap 자료구조

    => 우선쉰위 큐 기능을 구현하기 위해 사용

* bisect: 이진 탐색(Binary Search) 기능 제공

* collections: deque, counter등의 유용한 자료구조 포함

* math: 필수적인 수학 기능제공

    => 팩토리얼, 제곱근, 최대공약수, 삼각함수, 파이(PI)와 같은 상수 포함
